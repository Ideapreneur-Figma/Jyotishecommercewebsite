# App Icons

Place your app icons in this directory with the following sizes:

## Required Icon Sizes

- icon-72x72.png (72x72 pixels)
- icon-96x96.png (96x96 pixels)
- icon-128x128.png (128x128 pixels)
- icon-144x144.png (144x144 pixels)
- icon-152x152.png (152x152 pixels)
- icon-192x192.png (192x192 pixels)
- icon-384x384.png (384x384 pixels)
- icon-512x512.png (512x512 pixels)

## Design Guidelines

### Colors
- Primary: Orange (#ea580c)
- Background: Light Orange (#fff7ed)
- Icon: Om symbol (ğŸ•‰ï¸) or custom Jyotish logo

### Specifications
- **Format**: PNG with transparency
- **Safe Zone**: Keep important elements within 80% of icon area
- **Padding**: 10% padding on all sides
- **Background**: Use solid color or gradient
- **Maskable**: Design should work with Android adaptive icons

## Icon Design Tips

1. **Simple & Clear**: Icon should be recognizable at small sizes
2. **Consistent Branding**: Use brand colors and symbols
3. **High Contrast**: Ensure visibility on all backgrounds
4. **Test on Device**: Preview on actual mobile devices
5. **Adaptive Ready**: Design should work when cropped to circle
